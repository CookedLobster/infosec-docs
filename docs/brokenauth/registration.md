:::info Scenario

There is an Already Registered User **Dave.** 

:::


<br/>


- To try to gain Access to hes Account we can try to Register as <b style={{ color: 'Grey' }}>_</b><b style={{ color: 'Chartreuse' }}>Dave</b>
- It will actually register a new user but that user will have the same rights as normal Dave. That new user will also be able to see all the content presented under the user Dave.

<br/>




- `( _ ):` Blank Space before Entering the Username

```js
┌──────────┐
| USERNAME | -► _Dave
|──────────|
| PASSWORD | -► **********
└──────────┘
```
