:::info [Web Vulnerabilities](https://infosec-docs.pages.dev/category/web-vulnerabilities)

:::

Module **Injection** includes Vulnerabilities such as `LFI-RFI [Local-Remote File Inclusion]` as well as Pratical Examples In The Wild. `SQLi [SQL Injection]` both **Error Based - Blind** there are some Examples about **Authentication Bypass** etc. `SSTI [Server Side Template Injection]` as of now it not very well Documented.

Module **Broken Authentication** includes old `Re-Registration` Bypasses **Logic Flaw - Brute-Force.** `JWT [Json Web Token]` contains and Exploitation example of how the Process is going to Work.

Module **XML External Entity** explains how the  **XXE Exploitation** Process will Work. `SSRF [Server-Side Request Forgery]` contains a Pratical example on a Support Website.

Module **XSS Cross-Site Scripting** contains a lot of examples of how `XSS - Blind XSS` Works. `IDOR [Insecure Direct Object Reference]` is a Work In Progress.

Module **Upload Vulnerabilities** has escaping examples about `[Server-Client] Filtering`.
